# 💌 Carta Interactiva para una Princesa Especial

## Descripción del Proyecto
Este es un proyecto web interactivo que consiste en una **carta digital animada** diseñada especialmente para dedicar a una amiga muy especial, a quien se le llama cariñosamente "princesa".

La carta incluye:
- 🎨 Un diseño visual atractivo con colores suaves y elegantes
- 👑 Una corona animada como elemento central
- 💖 Corazones flotantes y efectos de partículas
- ✨ Animaciones interactivas al pasar el cursor
- 🎁 Un mensaje sorpresa que se revela con un botón
- 💕 Una barra de amor que llega hasta "infinito"
- 🌟 Lluvia de emojis celebrando el cariño

## Cómo Usar
1. Abre el archivo `carta-princesa.html` en cualquier navegador web moderno (Chrome, Firefox, Safari, Edge).
2. Disfruta de las animaciones automáticas.
3. Haz clic en el botón "Ver mensaje especial" para revelar la dedicatoria.
4. Mueve el cursor por la pantalla para ver los efectos de sparkles.

## Tecnologías Utilizadas
- HTML5
- CSS3 (con animaciones y gradientes)
- JavaScript (para la interactividad y efectos)

## Personalización
Puedes editar el archivo `carta-princesa.html` para cambiar:
- El texto de la dedicatoria
- Los colores del diseño
- La duración de las animaciones
- Los emojis que aparecen en la lluvia

¡Es perfecto para sorprender a esa persona tan especial de una manera única y creativa! 👑💕
